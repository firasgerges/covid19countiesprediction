------------------------Notes for Data files Format------------------------------------
1-train and test data files must be comma separated, and class label must be the final column
2- Attribute file must be in the same format as the names file of C4.5:
	2a-First line is all the class labels separated by a comma.
	2b-Each following line correspond to an attribute, Attribute Name: value.
	2c- value is either 'continious' or comma separated list of all the possible values.
3- No ID fields must be presented. all data must be used for training and testing model.
4- No header must be presented in data files
